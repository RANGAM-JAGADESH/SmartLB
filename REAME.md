# SmartLB

A Production-Inspired Intelligent HTTP Load Balancer built from scratch in Python.

## Features (Planned)

- Reverse Proxy
- Round Robin
- Weighted Round Robin
- Least Connections
- Least Latency
- IP Hash
- Sticky Sessions
- Health Checks
- Circuit Breaker
- Retry Logic
- Rate Limiting
- Metrics
- Dashboard
- Logging
- Docker Support (Final Phase)

## Tech Stack

- Python
- FastAPI
- HTTPX
- asyncio
- aiohttp
